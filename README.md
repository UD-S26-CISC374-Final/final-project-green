# Blurb

You work at a top secret spy agency. Each day you are given a program, and must use it to identify whether or not your co-workers are who they say they are, or if they are an impostor trying to sneak in. You must trace a the given program to determine if you should let the supposed co-worker in, or if you should have security get rid of them. You will also have a variety of tasks you must complete to help the agency track down the rival spies trying to sneak their way in.

# Basic Instructions

Each day you will be given a program. At the top of this program you will have each codename with a value set to one of your co-worker's real names. At the end of this program there are print statements that print the value of the codenames. Your job is to trace the program to see what name is printed and see if it matches the name on the ID card that your co-worker will give you. You will also occasionally have one or more tasks you must complete alongside your regular duties.
